##アプリケーション名
「ポートフォリオフォーラム」

##アプリケーション概要
ポートフォリオを投稿・閲覧するサイトです

##URL
https://port-folio-forum-35dcc.firebaseapp.com/

##使用技術
-HTML&CSS
-JavaScript
-Vue.js
-Vuetify
-Nuxt.js
-firebase

##機能一覧
-ユーザー登録
-ログイン機能
-ログインの常態化（ゲストユーザーに常態化機能はなし）
-投稿機能
-投稿のいいね機能
-投稿の編集機能
-投稿の削除機能
-ユーザー情報の編集機能
-ログアウト機能
